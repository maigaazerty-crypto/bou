Questions Fréquentes

Quand le jeu sera-t-il disponible ?

+

Nous travaillons actuellement sur la version bêta qui sera disponible dans les prochains mois. Inscrivez-vous pour être parmi les premiers à tester le jeu !

Sur quelles plateformes sera disponible le jeu ?

+

Cheick's World sera disponible sur PC, PlayStation, Xbox et Nintendo Switch. Une version mobile est également prévue pour l'avenir.

Le jeu sera-t-il gratuit ou payant ?

+

Cheick's World sera un jeu premium avec un achat unique. Nous proposons un contenu riche et une expérience complète sans microtransactions intrusives.
